## 開発&デプロイ
1. プロジェクトのクローン
```bash
git clone https://github.com/302ai/302_red_packet_generation
cd 302_red_packet_generation
```

2. 依存関係のインストール
```bash
pnpm install
```

3. 環境設定
```bash
cp .env.example .env.local
```
必要に応じて`.env.local`の環境変数を修正してください。

4. 開発サーバーの起動
```bash
pnpm dev
```

5. プロダクションビルド
```bash
pnpm build
pnpm start
```
